# **Image viewer**

## **Description**

:material-menu-open: **Visualization  â†’ Image viewer**

This visualization module provides an overview of imaging data.

![viewer](image_viewer.png)

The interactive imaging plot (left) allows to choose any pixel and explore related spectrum (bottom right). Vice versa, when selecting a single from the spectrum, the corresponding image will be generated. In *Spectrum options* menu, the user can choose a feature from a feature list to be depicted on a scan (prior feature detection is required).

The top right panel gives information about file and imaging parameters. The *Settings* menu below can be used to change the imaging parameters.

## **Parameters and Settings**
### Scan selection ###
Filter scans

### m/z range ###
Define m/z extraction window.

### Mobility ###
Define mobility extraction window if available.

### Transform pain scale ###
Apply a paint scale transformation for the image plot.

### Normalize images ###
Apply image normalization.
